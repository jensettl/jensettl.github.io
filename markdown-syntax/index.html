    Markdown Syntax

    Written by Jens on 21 Aug 2025

    This markdown cheat sheet covers just the basic syntax of markdown. For more information search the web.

    Summary

    • Basic
    • Extended Syntax

    This Markdown cheat sheet provides a quick overview of all the Markdown syntax elements. It can’t cover every edge case

    Basic

    These are the elements outlined in John Gruber’s original design document. All Markdown applications support these elements.

    Element Markdown Syntax
    Headinng # H1
    ## H2
    ### H3
    Bold Text ** Text **
    Italic * Text *
    Blockquote > Text
    Ordered List 1. First
    2. Second
    3. Third
    Unordered List - First
    - Second
    - Third
    Code `Code`
    Horizontal Rule —
    Link [Text] (URL)
    Image ! [Text] (URL)


    Extended Syntax

    These elements extend the basic syntax by adding additional features. Not all Markdown applications support these elements.

    Element Markdown Syntax
    Table | Header 1 | Header 2 |
    | —- | ———– |
    | Cell 1 | Cell 2 |
    Fenced Code Block ```
    {
    “firstName”: “John”,
    “lastName”: “Smith”,
    “age”: 25
    }
    ```
    Footnote Here’s a sentence with a footnote. [^1]
    [^1]: Another sentence with footnote.
    Striketrough ~~The world is flat.~~
    Escape Symbol \
